# SSL

## OpenSSL

Check if you already have OpenSSL installed:

```shell
openssl version
```

### ⚠️ If error

#### `Mac - Brew`

```shell
brew install openssl
```

#### `Otherwise`

Please follow instructions [here](https://github.com/openssl/openssl) to install it.

## Run

### `Linux/MacOS`

```shell
chmod +x ssl.sh
./ssl.sh
```
