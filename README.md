## Instructions

* clone the project

### Running greet project
```sh
make greet
```

First terminal
```sh
./bin/greet/server
```

Second terminal
```sh
./bin/greet/client
```

### Running calculator project
```sh
make calculator
```

First terminal
```sh
./bin/calculator/server
```

Second terminal
```sh
./bin/calculator/client
```
